Project description
PyOpenGraph is a library written in Python for parsing Open Graph protocol information from web sites.
Learn more about the protocol at
http://opengraphprotocoLorg
Installation
To install, download the archive at http://pypipython.org/pypi/PyOpenGraph and decompress, run python setup.py
install
Usage
‘import PyOpenGraph
og = PyOpenGraph( "http: //mww. rottentomatoes .com/m/10011268-oceans/")
print og.metadata # => {'url': ‘http: //mm.rottentomatoes.com/m/10011268-oceans/', 'site_name’
print og.metadata['title'] # => Oceans
og-is_valid() # => return True or False
